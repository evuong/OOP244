However many that the programmer wants for that family or given name +1. The +1 is to account for the null character at the end of the string.

2) Your class declares two member functions named set(...). In C, this would generate an error. Name the feature of C++ that allows this.

The C++ feature that allows this is called function overloading. It works because the program requires different parameters allowing it to know which instance of the function to call.
